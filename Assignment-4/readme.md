# Mintlify Platform Re-creation

This project is a high-fidelity recreation of the Mintlify landing page, focusing on modern UI/UX patterns, dark-themed aesthetics.

## Sections Recreated

- **Navigation Header**: Transparent sticky header with glassmorphism effects and call-to-action buttons.
- **Hero Section**: Features a dynamic background, "New" feature pill, main value proposition, and an email capture form.
- **Company Showcase**: A grid-based section displaying partner logos with custom SVG styling.
- **Enterprise Features**: A two-column layout highlighting platform capabilities with custom CSS iconography.
- **Intelligence Section**: A grid layout showcasing AI integration features using modern CSS Grid.
- **Customer Testimonials**: A dedicated space for social proof and customer success stories.
- **Advantage/CTA Section**: A final conversion section with a radial gradient background and feature highlights.
- **Site Footer**: A comprehensive multi-column footer with social links, security badges, and status indicators.

## Design Specifications

### Fonts
- **Primary Font**: [Manrope](https://fonts.google.com/specimen/Manrope) (Weights: 300, 400, 500, 600, 700, 800)
- **System Fallbacks**: system-ui, -apple-system, Segoe UI, Arial, sans-serif.

### Colors
- **Background Main**: `#080808` (Deep Black)
- **Text Main**: `#e6f2ff` (Soft Blue-White)
- **Accent Green**: `#2ddf8f` (Used for labels and icons)
- **Border/Stroke**: `rgba(255, 255, 255, 0.18)` (Subtle glass effect)
- **Button Primary**: `#ffffff` with `#0b1220` text.
